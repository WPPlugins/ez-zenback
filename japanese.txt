はじめに

「EZ zenback」は、「zenback」の導入を簡単にするプラグインです。
「zenbackコード」や「zenbackタグ」の挿入が設定画面から簡単に行えます。
また、任意のHTMLやJavaスクリプトを本文の前後に挿入する機能も備えています。

zenbackは、Six Aaprt, Ltd.の登録商標です。

特徴
・「zenbackコード」と「zenbackタグ」を簡単にインストール。
・任意のHTML、JavaScriptを簡単に本文、抜粋の前後に挿入。
・zenbackは、コメントの前または後ろに表示できる他、ウィジェットとしても登録可能。
・ショートコードのよるzenbackの表示もサポート。
・スタイルシートによるデザインのカスタマイズが可能。
・管理画面やメッセージは、英語と日本語(UTF-8)に対応。

注意
・このプラグインは、ご利用のテーマによっては正常に動作しません。
・このプラグインは、標準のコメントシステムに依存しています。コメント欄が許可されていないページでも「zenback」を表示出来ますが、標準のコメントが別のコメントシステムで置き換えられている場合は、「zenback」を表示出来ません。
・コメントシステムとして「DISQUS」を導入しているブログに「EZ zenback」をインストールする場合は、「Disqus Comment System」プラグインの代わりに「Disqus Comment System for EZ zenback」プラグインをインストールして下さい。

Disqus Comment System for EZ zenback：http://www.near-mint.com/blog/software/disqus-comment-system-for-ez-zenback


1.動作環境

・WordPress バージョン3.0 以上
・WordPress バージョン2.8 以上 + Disqus Comment System for EZ zenback


2.インストール

ダウンロードしたファイルを解凍し、「ez-zenback」フォルダごと、WordPressのプラグインディレクトリ（wp-content/plugins/）にアップロードします。
プラグインを有効化します。
WordPressの管理画面の「設定」の中の「EZ zenback」という項目で設定を行います。


3.アップデート

WordPressの管理画面で、「EZ zenback」を停止し、インストールの要領でファイルをサーバーにアップロード、上書きした後、再度、有効化する。


4.アンインストール

WordPressの管理画面で、まずプラグインを停止し、削除を実行する。
